Professional ticketing application for a company. It allows the resolution of incidents or breakdowns between a customer service and a technical support department. It was developed using CodeIgniter PHP framework and JavaScript. It has a responsive design for PC and mobile using Bootstrap. You need a MySQL database with a series of related tables to store the information and connect via PDO.

The customer service department will create notifications through forms that will appear in a control panel from which SAT can assign and process or resolve them. It also includes other sections for managing a business such as inventory, logistics, schedules, supervision, etc. and general purpose options such as user management, account, profile, etc.

It allows for the automatic sending of incidents alerts via email (CodeIgniter mailiing, Swiftmailer or PHPMailer) and Telegram (API). Extras: PDF through FPDF and Excel with PHPExcel generation.
